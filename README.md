# Warhammer 40,000 Chronological Reading Order

A comprehensive, lore-accurate chronological list of Warhammer 40,000 (Black Library) works, ordered by **in-universe events** rather than publication date.

This repository provides a **complete, non-simplified timeline**, including novels, novellas, short stories, and anthologies, with maximum preservation of narrative continuity and minimal spoilers.

---

## Features

* Full coverage of:

  * Horus Heresy (including all major arcs)
  * Siege of Terra (**including novellas**)
  * Post-Heresy (M31–M32)
  * War of the Beast
  * Classic 40k (M41)
  * Indomitus Era (M42)
* Includes:

  * Novels
  * Novellas
  * Short stories
  * Anthologies
* Long-running series are **fully expanded into exact reading order**
* Chronology based strictly on **in-universe events**
* **No content loss** (short stories and novellas are not omitted)
* **Chaos and Xenos content fully integrated**
* Separate blocks for:

  * Non-linear timelines (e.g. Necrons)
  * Weakly connected factions (e.g. T’au)
* **Official Russian titles** included where available
* **Russian descriptions** for each entry

---

## Data Structure

Main file:

```bash
warhammer_40k_chronology_ru_desc_filled_v3.xlsx
```

Each row contains:

* Title (EN)
* Title (RU, if official)
* Series / Arc
* Timeline position (M30–M42)
* Faction (where applicable)
* Type (novel / novella / short story / anthology)
* Short description (RU)

---

## Key Consistency Rules

* Critical narrative ordering preserved:

  * *The First Heretic → A Thousand Sons → Prospero Burns*
* No duplicated entries (e.g. *Vengeful Spirit*)
* No overlapping arcs:

  * Calth arc
  * Imperium Secundus
  * White Scars arc
* Siege of Terra novellas are **explicitly included and positioned correctly**
* Non-linear works (e.g. *The Infinite and the Divine*) are isolated into faction blocks

---

## Notes

* Some works (especially anthologies and Necron stories) have **approximate placement** due to non-linear timelines
* Codices, campaign books, and artbooks are intentionally excluded
* Russian titles are only included when **confirmed official translations exist**

---

## Goal

To provide a **reference-grade chronological reading order** that:

* Preserves narrative flow
* Avoids timeline confusion
* Minimizes spoilers
* Reflects canon as accurately as possible

# Хронологический порядок чтения Warhammer 40,000

Полная и максимально точная хронологическая таблица произведений Warhammer 40,000 (Black Library), отсортированная по **событиям внутри вселенной**, а не по дате выхода.

Репозиторий содержит **полную версию без упрощений**, включая романы, новеллы, рассказы и антологии, с сохранением логики повествования и минимизацией спойлеров.

---

## Особенности

* Полный охват:

  * Ересь Хоруса (включая все ключевые арки)
  * Осада Терры (**включая новеллы**)
  * Пост-ересь (M31–M32)
  * Эра Зверя
  * Классический 40k (M41)
  * Эра Индомитус (M42)
* Включает:

  * Романы
  * Новеллы
  * Рассказы
  * Антологии
* Длинные серии **развернуты в точный порядок чтения**
* Хронология строго по **событиям в лоре**
* **Ничего не вырезано** (рассказы и новеллы сохранены)
* Полностью добавлены:

  * Хаос
  * Ксеносы
* Отдельные блоки для:

  * Нелинейных таймлайнов (например, некроны)
  * Слабо связанных фракций (например, тау)
* Добавлены **официальные русские названия**
* Для каждого произведения есть **описание**

---

## Структура данных

Основной файл:

```bash
warhammer_40k_chronology_ru_desc_filled_v3.xlsx
```

Каждая строка содержит:

* Название (EN)
* Название (RU, если официальное)
* Серия / арка
* Позиция в таймлайне (M30–M42)
* Фракция (если применимо)
* Тип (роман / новелла / рассказ / антология)
* Краткое описание

---

## Правила целостности

* Сохранена ключевая последовательность:

  * *The First Heretic → A Thousand Sons → Prospero Burns*
* Нет дублей (например, *Vengeful Spirit*)
* Нет пересечений арок:

  * Калт
  * Imperium Secundus
  * Белые Шрамы
* Новеллы Осады Терры **не потеряны и стоят на своих местах**
* Нелинейные произведения (например, *The Infinite and the Divine*) вынесены в отдельные блоки

---

## Примечания

* Некоторые произведения (особенно антологии и некроны) имеют **условную хронологию**
* Кодексы, кампейн-книги и артбуки исключены
* Русские названия добавлены только при наличии **официального перевода**

---

Создать **эталонный хронологический гайд по Warhammer 40k**, который:
