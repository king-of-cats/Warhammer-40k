# Warhammer 40,000 Chronological Reading Order

A complete, lore-accurate chronological list of Warhammer 40,000 (Black Library) publications, ordered by **in-universe events**, not by release date.

This project aims to provide the most precise reading order possible for those who want to experience the setting **without timeline confusion, major flashbacks, or unintended spoilers**.

---

## Features

* Full coverage of:

  * Horus Heresy
  * Siege of Terra
  * Post-Heresy (M31–M32)
  * War of the Beast
  * Classic 40k (M41)
  * Indomitus Era (M42)
* Includes:

  * Novels
  * Novellas
  * Short stories
  * Anthologies
* Long series are **fully expanded into correct internal order**
* Chronology based on **in-universe events**, not publication order
* **Official Russian titles** included where available
* Each entry contains a **brief description (RU)**

---

## Structure

The dataset is provided as an Excel file:

```
warhammer_40k_chronology_ru_desc.xlsx
```

Each row includes:

* Title (EN)
* Title (RU, if official)
* Series / Arc
* Timeline placement (e.g. M30, M31, M41, M42)
* Short description (RU)

---

## Notes

* Some works (especially anthologies and certain Necron stories) have **non-linear or uncertain chronology**. In such cases, placement is approximate but lore-consistent.
* Campaign books, codices, and artbooks are intentionally excluded.
* Where an official Russian translation is uncertain, the field is left blank.

---

## Goal

To create a **reference-grade chronological reading guide** for Warhammer 40k that:

* Preserves narrative flow
* Minimizes spoilers
* Reflects canon as accurately as possible

# Хронологический порядок чтения Warhammer 40,000

Полная хронологическая таблица произведений Warhammer 40,000 (Black Library), отсортированная по **внутриигровым событиям**, а не по дате выхода.

Проект предназначен для тех, кто хочет читать вселенную **последовательно, без путаницы во времени и лишних спойлеров**.

---

## Особенности

* Полный охват:

  * Ересь Хоруса
  * Осада Терры
  * Пост-ересь (M31–M32)
  * Эра Зверя
  * Классический 40k (M41)
  * Эра Индомитус (M42)
* Включает:

  * Романы
  * Новеллы
  * Рассказы
  * Антологии
* Длинные серии **развернуты по правильному порядку книг**
* Хронология основана на **событиях внутри лора**, а не на дате публикации
* Добавлены **официальные русские названия** (если существуют)
* Для каждого произведения есть **краткое описание**

---

## Структура

Данные представлены в Excel-файле:

```
warhammer_40k_chronology_ru_desc.xlsx
```

Каждая строка содержит:

* Название (EN)
* Название (RU, если официальное)
* Серия / сюжетная арка
* Позиция в таймлайне (например M30, M31, M41, M42)
* Краткое описание

---

## Примечания

* Некоторые произведения (особенно антологии и истории про некронов) имеют **нелинейную или условную хронологию** — в таких случаях указана наиболее логичная позиция.
* Кодексы, кампейн-книги и артбуки исключены.
* Если официального перевода нет — поле оставлено пустым.

---

## Цель

Создать **эталонный хронологический гайд по чтению Warhammer 40k**, который:

* Сохраняет целостность повествования
* Минимизирует спойлеры
* Максимально точно отражает канон
